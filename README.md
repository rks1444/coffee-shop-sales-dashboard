# Coffee Shop Sales Dashboard (Excel)

## Project Overview
This project analyzes coffee shop sales data using Microsoft Excel.
An interactive dashboard was created to track sales performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

## Key Insights
- Different Coffee Size
- Best-selling products
- Peak sales hours
- Bills at Different Locations
- No. of Orders on Weekdays
